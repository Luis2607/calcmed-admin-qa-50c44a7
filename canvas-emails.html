Guia de Handoff Técnico para E-mails

Consolidação de boas práticas de compatibilidade (Outlook/Gmail) e integração no construtor do GoHighLevel (GHL).

1. Botão CTA Primário "Bulletproof" (Outlook-Safe)

O Outlook Desktop ignora paddings normais aplicados em links. Use esta tabela com tags VML no construtor de HTML:

<table role="presentation" border="0" cellpadding="0" cellspacing="0" style="margin: 0 auto;">
  <tr>
    <td align="center" bgcolor="#00B4D8" style="border-radius: 8px; padding: 14px 24px;">
      <!--[if mso]>
      <v:roundrect xmlns:v="urn:schemas-microsoft-com:vml" xmlns:w="urn:schemas-microsoft-com:vml" href="https://calcmed.design" style="height:48px;v-text-anchor:middle;width:280px;" arcsize="12%" stroke="f" fillcolor="#00B4D8">
        <w:anchorlock/>
        <center style="color:#ffffff;font-family:sans-serif;font-size:16px;font-weight:bold;">Acessar no Plantão</center>
      </v:roundrect>
      <![endif]-->
      <a href="https://calcmed.design" style="font-size: 16px; font-family: 'Inter', Arial, sans-serif; font-weight: 600; color: #FFFFFF; text-decoration: none; display: inline-block; -webkit-text-size-adjust:none; mso-hide:all;">
        Acessar no Plantão <span style="font-family: Arial, sans-serif; font-weight: bold; margin-left: 8px;">&rsaquo;</span>
      </a>
    </td>
  </tr>
</table>

2. Abas Clínicas Interativas (CSS Checkbox Hack)

Clientes de e-mail modernos (Apple Mail, iOS) suportam alternância de abas sem carregar scripts. O fallback para clientes antigos exibe as duas abas abertas sequencialmente.

<!-- Radio Triggers -->
<input type="radio" id="tab-adulto" name="clinico-tabs" checked style="display:none !important;">
<input type="radio" id="tab-pediatria" name="clinico-tabs" style="display:none !important;">

<!-- Tab labels -->
<label for="tab-adulto" class="aba-tab-btn" id="label-adulto" style="display:inline-block; padding:8px 16px;">Adulto</label>
<label for="tab-pediatria" class="aba-tab-btn" id="label-pediatria" style="display:inline-block; padding:8px 16px;">Pediátrico</label>
Componente
  • Largura: 600px
  • Bordas: 8px
